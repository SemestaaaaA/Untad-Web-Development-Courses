About HTML

HTML (HyperText Markup Language) is the standard language used to create web pages. It provides the structure and layout for content on the web, allowing developers to define elements such as text, images, links, forms, and other media.

HTML History

In 1980, physicist Tim Berners-Lee, a contractor at CERN, proposed and prototyped ENQUIRE, a system for CERN researchers to use and share documents. In 1989, Berners-Lee wrote a memo proposing an Internet-based hypertext system. Berners-Lee specified HTML and wrote the browser and server software in late 1990. That year, Berners-Lee and CERN data systems engineer Robert Cailliau collaborated on a joint request for funding, but the project was not formally adopted by CERN. In his personal notes of 1990, Berners-Lee listed "some of the many areas in which hypertext is used"; an encyclopedia is the first entry.

HTML Characteristics

  • The file extension is "abc.html"
  • Platform independent
  • Consists of opening and closing tags
  • Tags pair and nest with each other
  • Semantic elements
  • Offline capabilities

HTML Structure

  • Document Information

    • The <html> element is the root element of an HTML page
    • The <!DOCTYPE html> declaration defines that this document is an HTML5 document

  • Document Header

    • The <head> element contains meta information about the HTML page
    • The <title> defines the title displayed on the top of your browser when you visit a website

  • Document Body

    • The <body> element defines the document's body and contains all visible content like headings, paragraphs, images, etc.
    • The <h1> element defines a large heading
    • The <p> element defines a paragraph
    • The <style> tag allows us to insert styling into our web pages

Standard HTML document structure

<html>

<head>

.....

</head>

<body>

.....Your Codes.....

</body>

</html>

  • <p>

  • <hr>
    horizontal line

    line

- Font Tag

  • Example:
      • <Font color=gray size=7 face="arial" test font>
  • <b> bold
  • <i> italic
  • <u> underline
  • <sub> subscript
  • <sup> superscript

- Image Tag

      Example : <img src="../img/stamp.png" >
  • src = Specifies the path to the image
  • width = Specifies the width of an image
  • height = Specifies the height of an image
  • alt = Specifies alternate text for the image if it cannot be displayed

- Anchor/Hyperlink Tag

To access a specific part inside the website:

  • Marker on the page
      • <a id="above">this is the first alinea
  • Link to the same file
      • <a href"#above">back to top

- Comments Tag

The comment tag is used to insert comments in the source code. Comments are not displayed in browsers.
